Easy

1.Материнская плата - объединяет все компоненты компьютера.
Процессор - занимается всеми задачами ПК, следует инструкциям которые мы ему даем, например пишем вывести, что то на монитор и он это выводит.
Оперативная память - в оперативной памяти и хранятся все инструкции которые мы даем нашему ПК.
Долгосрочная память - нужна для хранения информации больших объемов.

2. Stack предназначен для статического распределения данных, а Heap для динамического.

3. В Java существует вещественный, целочисленный, ссылочный, логический и символьный типы данных.

4. Отличия в том, что примитивные тип хранят значение напрямую, а ссылочные типы хранят адреса объектов в памяти.

5. Переменнная - область в памяти, предназначенная для хранения определенного типа данных.

6. ! - отрицательный оператор, & - логическое умножение, | - логическое сложение.

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scan = new Scanner(System.in);
        System.out.println("Введите 3 вещественных числа: ");

        double a = scan.nextDouble();
        double b = scan.nextDouble();
        double c = scan.nextDouble();

        double s = a + b + c;
        System.out.println("Сумма этих чисел: " + s);

        double j = a * b * c;
        System.out.println("Произведение этих чисел: " + j);
    }
}
